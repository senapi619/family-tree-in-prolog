# Author
Bigyan Devkota

## Assignment 8 - MSCS-632-A01
Advanced Programming Languages - First Bi-term 2026

## Description
Prolog program implementing a three-generation family tree with logical inference rules for parent, sibling, cousin, ancestor, and descendant relationships.

## Files
- `family_tree.pl` - Main program with facts, rules, and recursive predicates
- `sample_queries.pl` - Sample queries with expected output

## How to Run
1. Open SWI-Prolog
2. Load the file: `?- [family_tree].`
3. Run queries as shown in `sample_queries.pl`
